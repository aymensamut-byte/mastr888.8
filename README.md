# 3D Fighting Game - Tycoon Edition 🎮

لعبة قتال ثلاثية الأبعاد مبنية من الصفر باستخدام Unity!

## ✨ الميزات

### 🎮 نظام اللعب
- **حركة ديناميكية**: WASD للحركة، Space للقفز
- **نظام قتال**: Left Click للكم مع كشف اصطدام دقيق
- **ذكاء اصطناعي للعدو**: مطاردة، هجوم، وحركات عشوائية
- **أشرطة صحة**: عرض مباشر لصحة اللاعب والعدو

### 👾 الشخصيات
- **اللاعب (أزرق)**: شكل إنساني أنيق بألوان أزرق مميزة
- **العدو (أحمر)**: شكل مختلف تماماً مع قرون سوداء وعيون حمراء

### 🤖 نظام الذكاء الاصطناعي
- كشف اللاعب تلقائياً عند الاقتراب
- مطاردة ذكية مع دوران سلس
- هجوم منتظم عند الاقتراب
- قفز عشوائي أثناء المطاردة

### 🎨 الواجهة
- شرط صحة ديناميكي (أخضر -> أحمر)
- عرض قيم الصحة الحالية والقصوى
- رسالة نصر أو هزيمة

## 🚀 كيفية البدء

1. **فتح المشروع في Unity 2020+**

2. **إنشاء المشهد:**
   - أنشئ مشهد جديد
   - أضف إضاءة (Main Light + Ambient)
   - أضف أرضية (Plane مع Material)

3. **إنشاء اللاعب:**
   - Cube جديد → أضف `PlayerController` و `CharacterModel`
   - ضع `isPlayer = true` في CharacterModel
   - أضف Rigidbody و BoxCollider (عين Tag = "Player")
   - ضع الطبقة على "Player"

4. **إنشاء العدو:**
   - Cube جديد → أضف `EnemyController` و `CharacterModel`
   - ضع `isPlayer = false` في CharacterModel
   - أضف Rigidbody و BoxCollider (عين Tag = "Enemy")
   - عين Layer Mask للأرضية

5. **إنشاء الواجهة:**
   - أضف Canvas و UI Elements
   - وصل PlayerController و EnemyController في UIManager
